**验证清单**：

| 步骤     | 命令                  | 预期                                                         |
| :------- | :-------------------- | :----------------------------------------------------------- |
| 后端启动 | `npm run dev:backend` | 控制台输出 `启动于端口 18000`，项目根目录生成 `.backend-port` |
| 前端启动 | `npm run dev`         | 控制台输出 `后端 proxy 目标端口: 18000`                      |
| 连接状态 | 浏览器打开            | 左下角显示"已连接"                                           |
| 后端退出 | Ctrl+C                | `.backend-port` 文件自动删除                                 |
