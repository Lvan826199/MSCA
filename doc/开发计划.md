## 风险与应对

| 风险 | 影响 | 应对策略 |
| :--- | :---: | :--- |
| scrcpy 协议变更 | 高 | 锁定 scrcpy-server 版本，关注上游更新 |
| WebCodecs 浏览器兼容性 | 中 | 实现 MSE 降级方案 |
| WDA 签名过期（7 天） | 高 | 文档要求付费开发者账号，提供重签指引 |
| go-ios 对新 iOS 版本支持滞后 | 中 | 建立回退机制，关注社区更新 |
| 多设备并发性能瓶颈 | 中 | 背压控制 + 最大并发数限制 |
